# ARA-CheckIn: 실시간 동기화 기반 익명 출석 체크 시스템

> **"AI의 편리함 뒤에 숨은 백엔드의 본질을 탐구하다"**
>
> **데이터 정합성(Consistency)**과 **동시성 제어(Concurrency Control)**라는 백엔드의 핵심 가치를 실현하기 위해 AI와 협력하며 설계한 24시간 스프린트 프로젝트입니다.

---

## 🚀 프로젝트 개요 (Project Overview)
- **목적**: 배드민턴 클럽 회원들이 별도의 가입/로그인 절차 없이 링크 접속만으로 즉시 당일 출석을 확인하고 인원을 파악할 수 있는 **Zero-Friction UX** 시스템 구축.
- **핵심 가치**: 사용자 편의성 극대화, 실시간 데이터 동기화, 익명 환경에서의 데이터 무결성 보장.
- **개발 기간**: 2025.12.07 (1일 설계 및 구현)

## 🛠 Tech Stack
- **Backend**: Firebase Realtime Database (NoSQL, BaaS)
- **Frontend**: TypeScript, React, Tailwind CSS
- **Infrastructure**: Vite, Vercel (CI/CD)
- **AI Productivity**: ChatGPT, Cursor (Architecture Audit & Edge-case Testing)
